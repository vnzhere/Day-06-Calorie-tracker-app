# Day-06
📌 Calorie Tracker Web Application


📖 Project Description

Calorie Tracker is a simple and interactive web application designed to help users monitor their daily calorie intake. The application allows users to enter personal details such as height and weight, add meals with calorie values, track daily calorie goals, and view meal history.

The app focuses on an attractive GUI design, smooth navigation, and basic functional working as required for the academic assignment.

⸻

🎯 Objectives
	•	To design a multi-screen web application.
	•	To implement proper navigation between screens.
	•	To use interactive UI components like forms, buttons, progress bar, and lists.
	•	To store user data using LocalStorage for persistence.

⸻

🖥️ Features
	•	🏠 Dashboard
	•	Displays total calories consumed
	•	Shows calorie goal progress bar
	•	Displays BMI (based on height & weight)
	•	Reset button to clear data
	•	👤 Profile Screen
	•	Enter Name
	•	Enter Height (cm)
	•	Enter Weight (kg)
	•	Automatically calculates BMI
	•	🍽️ Add Meal Screen
	•	Enter meal name
	•	Enter calorie amount
	•	Adds meal to daily record
	•	📜 History Screen
	•	Displays list of added meals
	•	Shows calorie values
	•	Updates dynamically
	•	💾 LocalStorage Support
	•	Data remains saved even after refreshing the page

⸻

🔄 Navigation Flow

Dashboard
↓
Profile (Enter Height & Weight)
↓
Add Meal
↓
History
↓
Back to Dashboard

The Dashboard acts as the central screen from which all other screens are accessed.

⸻

🛠️ Technologies Used

	•	CSS
	•	JavaScript
	•	LocalStorage (for data persistence)
